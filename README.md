# 광주·전남 AI 여행 플래너 MVP

사용자의 여행 조건과 취향 선택을 바탕으로 광주·전남 여행 일정을 생성하는 정적 웹 MVP입니다.

## 구현 범위

- 광주, 담양, 여수, 순천, 목포 지역 샘플 데이터 포함
- 날짜, 인원, 동반 유형, 교통수단, 일정 강도, 아기·반려동물 조건 입력
- 6개 취향 토너먼트 선택
- 페르소나와 검색 키워드 생성
- 맑음, 비, 폭염·악천후 3개 플랜 생성
- 거리·시간·날씨·동반 조건 기반 시뮬레이션 보고서 표시
- 지도 API 없이 좌표 기반 마커 미리보기 제공

## 실행

브라우저에서 `index.html`을 열면 바로 실행됩니다.

## 배포

`.github/workflows/pages.yml`이 포함되어 있어 GitHub Pages를 GitHub Actions 방식으로 배포할 수 있습니다.
